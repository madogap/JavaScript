Construa uma página web que permita cadastrar desenvolveodes, suas teconlogias em tempos de experiência
=A página deve ter um input pra o nome completo do desenvolvedor e um botão para adiconar uma nova teconlogias
=Ao clicar no botão um evento deve ser disparado criando no formulário uma nova linha de inputs contendo:
:Um campo de texto para o nome da tecnologia
:Um  grupo de botões tipo "Radio" com opções de tempo de experiência (por exemplo - "0-2 anos", 3-4 e 5)
:Um botão de remover que deve excluir essa linha
:O formulário também deve ter um botão de cadastrar que salva as informações do dev em um array e limpa o formulário. Para testar 
o funcionamento você deve pode utilizar um simples console.log()
Eventos adicionados via JavaScript e a página deve funcionar sem acionar um recarregamento.
